КОМАНДЫ ДЛЯ ВОССТАНОВЛЕНИЯ:
---------------------------
1. Остановить сервисы:
   sudo systemctl stop quantum-nexus-web
   sudo systemctl stop quantum-nexus-bot

2. Загрузить файлы на сервер:
   scp -r quantum-nexus root@your-server:/root/

3. Установить права:
   chmod +x /root/quantum-nexus/*.sh
   chmod 644 /root/quantum-nexus/*.py

4. Обновить веб-файлы:
   sudo cp /root/quantum-nexus/web_app.html /var/www/quantum-nexus/
   sudo cp /root/quantum-nexus/admin.html /var/www/quantum-nexus/

5. Перезапустить сервисы:
   sudo systemctl start quantum-nexus-web
   sudo systemctl start quantum-nexus-bot
