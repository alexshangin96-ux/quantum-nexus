Команды для обновления админ-панели Quantum Nexus:

=== Вариант 1: Быстрое обновление (рекомендуется) ===

ssh root@ваш_IP
cd /root/quantum-nexus
git pull origin main
cp admin.html /var/www/quantum-nexus/
systemctl restart quantum-nexus-web

=== Вариант 2: Через готовый скрипт ===

ssh root@ваш_IP "cd /root/quantum-nexus && chmod +x UPDATE_SERVER.sh && ./UPDATE_SERVER.sh"

=== Вариант 3: Быстрое обновление в одну строку ===

ssh root@ваш_IP "cd /root/quantum-nexus && git pull origin main && cp admin.html /var/www/quantum-nexus/ && systemctl restart quantum-nexus-web"
